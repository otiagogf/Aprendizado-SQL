# Where mais like

## Filtro especial para textos

```sql
-- BUSCANDO CÉLULA COM O VALOR EXATO
SELECT 
    *
FROM
    DIMPRODUCT
WHERE
    PRODUCTNAME LIKE 'MP3'
```

```sql
-- BUSCANDO CÉLULA QUE CONTÉM MP3
SELECT 
    *
FROM
    DIMPRODUCT
WHERE
    PRODUCTNAME LIKE '%MP3%'
```

```sql
-- PARA BUSCAR MP3 NO FINAL DA CÉLULA % NO FINAL 
-- PARA BUSCAR MP3 NO FINAL DA CÉLULA % NO INÍCIO
SELECT 
    *
FROM
    DIMPRODUCT
WHERE
    PRODUCTNAME LIKE 'MP3%'
--  PRODUCTNAME LIKE '%MP3'
```
